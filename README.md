# Android Java – Property Tax Calculator

## 📖 Description
This Android application calculates property tax based on three key pieces of information:

- Property area (in m²)
- Number of rooms
- Presence of a swimming pool

The user enters the information into the application and presses the **Calculate** button to display the total tax amount.

## ⚙️ Features
- **EditText**: Input for area and number of rooms
- **CheckBox**: Option to indicate swimming pool presence
- **Button**: Triggers tax calculation
- **TextView**: Displays the calculation result with detailed breakdown:
  - Base Tax (Area × 2 DH)
  - Supplementary Tax (Rooms × 50 DH + Pool: 100 DH)
  - Total Tax
